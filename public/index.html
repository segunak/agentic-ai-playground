From Acronyms to Applications

What is this?

This is the API backend for the Agentic AI: From Acronyms to Applications workshop. It powers an AI agent that can autonomously decide when to use tools like checking the weather, fetching fun facts, and sharing Charlotte's best cinnamon roll spots.

How do I use it?

The interactive playground is embedded directly in the workshop course on VS Code for Education. If you're taking the workshop, open the course link I shared to get started. No GitHub account required, just a Microsoft account. Read my companion article for the full writeup.
